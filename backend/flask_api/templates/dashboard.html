Welcome to Fake Job Detection System

Explore Dataset

Explore Dataset provides a complete overview of the job posting data used in this project. It displays the total number of jobs, real jobs, and fake jobs, along with dataset statistics such as data distribution and key attributes.
 View Algorithms

View Algorithms explains the machine learning and deep learning models used for fake job detection. The algorithms implemented include Logistic Regression, Random Forest, CNN, and LSTM.
 Compare Models

Compare Models presents a comparative analysis of all implemented algorithms. It evaluates each model based on accuracy, precision, recall, F1-score, and AUC values.
 Predict Job Fraud

Fake Job Detection allows users to check whether a job posting is real or fraudulent by entering job details such as job title, description, and location.
